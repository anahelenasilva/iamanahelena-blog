### Rename branch
> `git branch -m <oldname> <newname>`


### Delete local branch
> `git branch -d <name>`

### Delete remote branch
> `git branch --delete <name>`

### Undo
> `git reset --hard`

### Undo unpushed
> `git reset --soft HEAD~1`
